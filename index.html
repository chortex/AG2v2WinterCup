General information

The tournament is held online in a professional Half Life game mode Adrenaline Gamer / OpenAG

Slag banned players are not allowed.

There were posted an important update in the servers section. Please check it out.

Before signing up to take part in a tournament make sure that you read the rules and you are able to show up for tournament games.

  • Requirements

Tournament consists of two stages: qualifiers stage (round robin) and olympic system with double elimination brackets

Qualification of participants is carried out randomly

Qualifiers (group stage)

Group stage is going with mode round robin (everyone vs everyone)

Every team picks home map for every single match from map list

3 points per win

1 point per draw

0 point per lose

Two teams with the highest points amount go to the next stage

If two teams have equal points amount team who won their face to face match is going forward

If their face to face match finished with score 1:1 team with better k:d is going forward

Brackets

This stage is double elimination with loser brackets

Every match must be played with mode bo3

Final must be played with mode bo5

Maps should be picked by elimination from a map list


Adrenaline Gamer 6.6 and OpenAG both are allowed

Any optional software what's giving you any kind of advantage (cheats) are forbidden

Spawn mines/satchels, teleport mines/satchels, ghost mines/satchels, path blocking with mines/satchels are not allowed. -5 frags will be deducted from that teams score per infraction

If a player does not spawn for 5 consecutive seconds upon death, 1 frag will be deduced from their final score. An additional frag will be deducted for each additional 5 seconds that passes without spawning

Fps limit is 144 frames per second. The violation will be punishable by the technical defeat of team on each map where used more than 144 fps

Allowed models are red and blue. Valve models can be used if opponent agrees. It's forbidden to switch models once game is agstarted

If both teams agreed any other game mode they are allowed to play it otherwise game mode should be set hlccl

Every player must have demos of his games and upload it if administrators requested

Winners team captain must upload screenshots of games to his steam profile or any other host

Any unsaid issues that arise will be dealt with by an admin


In qualifiers stage every team picks new home map for every single match from map list

Maps should be picked by elimination from a map list. Per one map for each team and third map is a decider map

Map pool might be changed after discussion with participants before 24th of January

! Important update !

Your map will be played in the opponents local region (server chosen by your team from server list) and your opponents map will be played in your local region (server chosen by them from server list). Rare exceptions are when teams will ping above 150 on servers. In these cases different regions can be used to adequately provide similar pings for more competitive game play.

Talking about playoffs where there is decider map it must be played in local region of team who has better k:d ratio after results home maps but server should be chosen by another team.

Any of the above need not apply if the two teams can come to an arrangement amongst themselves on which servers to use since teams might be regionally mixed. Only when an agreement cannot be made must the above be followed to ensure the best pings possible for the away team.

Any of the above need not apply if there is server in serverlist where teams have ping difference less than 70. In that case teams might be forcing to use a server with the smallest gap.
